## 交通基礎

- 新千歲機場到札幌：JR 快速 Airport 約 33-41 分鐘，白天班次密集。抵達日最省心的動線是入境後到地下 JR 站，搭車到 JR 札幌站，再依旅館位置轉地下鐵或步行。
- 札幌到小樽：JR 快速 Airport 或普通列車，最快約 33 分鐘，普通車約 46 分鐘。小樽市區可用「小樽站 -> 運河 -> 堺町通 -> 南小樽站」的步行動線。
- 札幌到旭川：JR 特急 Lilac/Kamui 約 1 小時 25 分鐘；旭川站再依當日行程轉巴士、計程車或市區交通。
- 札幌到登別：JR 特急往登別方向，再由 JR 登別站轉巴士到登別溫泉巴士總站。札幌同住旅館來回可行，但當天會比較長。
- 札幌市內：地下鐵南北線、東西線、東豐線加少量巴士即可完成。若旅館可選，札幌站、大通、薄野之間最方便。

## 住宿位置判斷：Hotel ONE'S RESIDENCE

Hotel ONE'S RESIDENCE 位在札幌中央區南 8 條西 2 丁目，靠近中島公園與薄野南側。以你們的行程來看，這個位置的優點是晚上吃飯、逛薄野、狸小路、二條市場都方便；缺點是去小樽、旭川、登別時，不是直接在 JR 札幌站旁邊，需要多抓一段地下鐵或計程車接駁時間。

### 旅館附近常用交通點

- 地下鐵南北線中島公園站：旅館主要地鐵站，步行約 3-5 分鐘。去札幌站、大通、薄野最直覺。
- 地下鐵東豐線豐水薄野站：步行約 7 分鐘左右。去福住站、羊之丘展望台方向會比較順。
- 薄野站 / 狸小路：步行可達，晚餐與夜間活動很方便。
- JR 札幌站：日歸小樽、旭川、登別、新千歲機場都會用到。建議從中島公園站搭南北線到札幌站，或多人帶行李時直接搭計程車。

### 對行程的影響

- Day 1 抵達：新千歲機場 -> JR 札幌站 -> 地下鐵南北線到中島公園站 -> 步行到旅館。若行李多或天氣熱，可從札幌站直接計程車到旅館。
- Day 2 小樽、Day 6 旭川、Day 7 登別：早上都先從中島公園站接到札幌站，再搭 JR 出發。這三天建議比列車發車時間多預留 20-30 分鐘。
- Day 3、Day 4、Day 5 札幌市內：旅館位置很好用，回程可彈性走薄野、狸小路、大通，不需要每天硬回札幌站。
- Day 8 返台：若行李多，建議旅館 -> 計程車到 JR 札幌站 -> JR 快速 Airport 到新千歲機場；或查是否有適合的機場巴士站點。

## 期間可能遇到的節慶與活動

### 札幌夏祭 / 大通啤酒花園 / 狸小路祭 / 北海盆踊

2026 年札幌夏祭主要在大通公園、狸小路、薄野一帶舉行。官方資料顯示，大通啤酒花園與狸小路祭為 7/23-8/18，北海盆踊為 8/13-8/16。你們 8/12-8/18 幾乎每天晚上都可以在大通、狸小路附近感受夏天祭典氣氛。  

安排建議：Day 1 抵達後先輕鬆逛大通與狸小路；Day 5 晚上特別保留給北海盆踊或夏祭收尾，因為 8/16 是 2026 年北海盆踊最後一晚。

### 小樽潮祭

小樽潮祭通常在 7 月最後的週五、週六、週日，會有港口活動、舞蹈、船隊、煙火。你們 8/13 到小樽時大概率已經錯過主祭，但小樽運河、堺町通、壽司街、玻璃與音樂盒街區本身仍是札幌出發最順的一日遊。

### 登別地獄谷鬼花火 / 登別地獄祭

鬼花火通常在夏季特定日期晚間於地獄谷展望台舉行，2026 年官方排程為 6 月、7 月與 9 月指定日，8 月沒有列入場次。2026 年登別地獄祭預計在 8/29-8/30，與你們 8/18 的登別日錯開。  

安排建議：Day 7 以地獄谷、溫泉街與伊達時代村為主，不把鬼花火或地獄祭列為必看活動。

## Day 1 - 2026/8/12（三）抵達新千歲機場、札幌初見

### 建議路線

13:35 抵達新千歲機場 -> 入境、領行李 -> JR 快速 Airport 到札幌 -> 地下鐵南北線到中島公園站或從札幌站計程車到 Hotel ONE'S RESIDENCE -> 飯店放行李/入住 -> 大通公園散步 -> 狸小路商店街 -> 薄野晚餐 -> 步行或地下鐵回飯店休息。

#### 1. 新千歲機場國際線航廈 / JR 新千歲機場站

新千歲機場本身很好逛，有伴手禮、甜點、拉麵道場、Royce 巧克力相關設施。抵達日若入境與行李領取很順，可以先買少量點心，但大件伴手禮建議最後一天再買，避免全程搬著走。

必吃：新千歲機場拉麵道場。若抵達後很餓，可先吃一碗北海道拉麵，但第一晚更建議留胃給札幌市區。

#### 2. JR 札幌站

札幌站是北海道最大交通樞紐之一，之後去小樽、旭川、登別與機場都會回到這裡。第一天先熟悉車站、地下街、轉地下鐵的位置，後面幾天會省很多力氣。

#### 3. Hotel ONE'S RESIDENCE / 中島公園站

抵達旅館後先放行李、確認入住方式、垃圾分類與是否可寄放行李。這間旅館靠近中島公園與薄野南側，晚上回程相對方便。

#### 4. 大通公園

大通公園是札幌市中心的長條型公園，夏天常有啤酒花園與活動攤位。你們抵達期間正好遇到札幌夏祭，大通公園很適合第一晚用輕鬆散步的方式進入北海道節奏。

必吃：大通啤酒花園/夏祭攤位。大人可體驗啤酒花園，青少年可吃烤物、玉米、馬鈴薯、甜點。

#### 5. 狸小路商店街 / 薄野商圈

狸小路是有屋頂的商店街，下雨也能逛，適合第一晚找藥妝、零食、輕鬆晚餐。薄野則是札幌夜間餐廳最密集的區域，離旅館也近。

必吃：湯咖哩、成吉思汗烤羊肉、札幌締めパフェ。湯咖哩蔬菜量多、辣度可調，青少年通常也容易接受；成吉思汗烤羊肉可放在這天或 Day 5；晚餐後的夜間聖代是札幌很有代表性的收尾。

## Day 2 - 2026/8/13（四）小樽一日遊

### 建議路線

Hotel ONE'S RESIDENCE -> 中島公園站 -> 札幌站 -> JR 小樽站 -> 三角市場早午餐 -> 小樽運河散步拍照 -> 壽司屋通午餐或下午早一點吃 -> 堺町通慢慢逛 -> 北一硝子、甜點店 -> 小樽音樂盒堂 -> 南小樽站搭 JR 回札幌 -> 地下鐵回中島公園站。  

如果想加碼余市 Nikka 威士忌蒸餾所：早上先從小樽轉 JR 到余市，參觀後再回小樽市區。但二大一青少年同行，且只有一天，我建議第一版先不加，避免小樽本體逛得太趕。

#### 1. Hotel ONE'S RESIDENCE -> 中島公園站 -> JR 札幌站

這天是第一個日歸城市移動日，建議提早出門，從中島公園站搭南北線到札幌站，再轉 JR 往小樽。早上多抓 20-30 分鐘，避免在札幌站找月台時壓力太大。

#### 2. JR 小樽站 / 三角市場

小樽站出來步行即可到三角市場，適合安排早午餐。市場空間不大，熱門店家仍可能排隊，但 8/13 是星期四，比週末友善。

必吃：三角市場海鮮丼。海膽、鮭魚卵、螃蟹類可視季節與預算選擇。

#### 3. 小樽運河

小樽是港町，最經典的是運河倉庫群、石造建築、玻璃工藝、音樂盒與海鮮。小樽運河適合白天與傍晚各看一次，白天能看倉庫與水道，傍晚燈亮後氣氛更好。

#### 4. 小樽壽司屋通

小樽以新鮮海產與壽司聞名，壽司屋通約有多間壽司店，適合作為正餐。若三角市場吃得早，這裡可安排午餐；若早午餐吃得飽，則改成下午較早的正餐。

必吃：小樽壽司。建議依當天排隊狀況選店，不必只鎖定單一名店。

#### 5. 堺町通 / 北一硝子

堺町通集中了甜點、玻璃、伴手禮店，從運河一路走到南小樽站剛好順路。北一硝子是小樽玻璃工藝代表之一，適合看玻璃器皿、燈飾與小物。

必吃：LeTAO 甜點。雙層起司蛋糕、霜淇淋、巧克力甜點很適合下午休息。

#### 6. 小樽音樂盒堂 / JR 南小樽站

小樽音樂盒堂主館是歷史建築，館內音樂盒種類非常多，外面的蒸汽鐘約每 15 分鐘響一次，是很容易拍照停留的點。逛完後從南小樽站回札幌，路線順而且不必折返小樽站。

必吃：若雞時代なると本店。小樽名物半身炸雞，喜歡炸物可列為晚餐或外帶候選。

## Day 3 - 2026/8/14（五）札幌市中心歷史與市場日

### 建議路線

Hotel ONE'S RESIDENCE -> 二條市場早餐（可步行或短程地下鐵/計程車） -> 大通公園與電視塔 -> 時計台 -> 紅磚廳舍外觀/周邊 -> 北海道大學散步 -> 札幌站周邊咖啡、購物 -> 晚上大通夏祭或薄野晚餐 -> 步行/地下鐵回旅館。

#### 1. Hotel ONE'S RESIDENCE -> 二條市場

這天設計成不用長距離移動的札幌市中心日，適合在前一天小樽步行後稍微放慢。從旅館到二條市場可步行、短程地下鐵或計程車，早上先吃海鮮很順。

必吃：二條市場海鮮丼。海膽、鮭魚卵、螃蟹類視季節與預算選擇。

#### 2. 大通公園 / 札幌電視塔

大通公園是札幌市中心主軸，電視塔則是很直覺的地標。這裡也和夏祭活動重疊，白天看城市景觀，晚上可再回來看活動氣氛。

#### 3. 札幌時計台

時計台是札幌代表性的歷史建築，停留時間不用長，適合從大通往札幌站方向串接。拍照後繼續往紅磚廳舍走，動線自然。

#### 4. 北海道廳舊本廳舍（紅磚廳舍）

紅磚廳舍是北海道開拓時代的代表性建築，周邊庭園也適合短暫散步。若遇到整修或部分區域限制，就以外觀與周邊停留為主。

#### 5. 北海道大學札幌校區

北海道大學校園很開闊，夏天綠意舒服，適合青少年走走拍照。若天氣熱，可縮短戶外停留，把午後移到札幌站地下街、百貨或 JR Tower 周邊。

必吃：札幌農學校餅乾、北海道牛奶甜點。適合在札幌站周邊找伴手禮試吃。

#### 6. 薄野 / 大通夜間

晚上回到大通夏祭或薄野晚餐，回旅館距離短，這天很適合安排比較晚的甜點或宵夜。

必吃：札幌味噌拉麵、湯咖哩。薄野拉麵橫丁方便；若 Day 1 沒吃湯咖哩，這天非常適合補上。

## Day 4 - 2026/8/15（六）白色戀人公園、円山、北海道神宮、藻岩山夜景

### 建議路線

Hotel ONE'S RESIDENCE -> 中島公園站搭地下鐵，於大通轉東西線到宮之澤站 -> 白色戀人公園 -> 回到東西線轉円山公園站 -> 北海道神宮、円山公園 -> 晚餐 -> 藻岩山夜景 -> 回旅館。  

若當天天氣不好：把藻岩山夜景改到 Day 5，Day 5 晚上較有彈性。

#### 1. Hotel ONE'S RESIDENCE -> 宮之澤站

從中島公園站搭地下鐵到大通轉東西線，再到宮之澤站。這段是札幌市內交通，不用拉行李，適合慢慢出門。

#### 2. 白色戀人公園

白色戀人公園是札幌代表性的甜點主題設施，可看、買、吃，也有甜點製作體驗與拍照點，對家庭旅遊很友善。從宮之澤站步行可到，交通簡單。

必吃：白色戀人公園限定甜點與霜淇淋。這裡很適合作為上午到中午的甜點休息點。

#### 3. 円山公園站 / 北海道神宮 / 円山公園

下午轉到円山區域，北海道神宮與円山公園綠意多，能感受札幌比較安靜的一面。若天氣炎熱，這段可走短一點，保留體力給晚上夜景。

必吃：円山周邊咖啡店、甜點店。這一帶比薄野安靜，適合下午坐下來休息。

#### 4. 晚餐

晚餐可選成吉思汗烤羊肉或海鮮居酒屋；若想靠近夜景動線，可先在薄野/大通用餐再去藻岩山。

#### 5. 藻岩山纜車 / 山頂展望台

藻岩山夜景視野開闊，是札幌很適合安排在旅程中段的亮點。若天氣不好、雲層低或雨勢明顯，就不要硬上山，改到 Day 5 晚上或取消。

## Day 5 - 2026/8/16（日）莫埃來沼公園、羊之丘展望台、札幌夏祭收尾

### 建議路線

Hotel ONE'S RESIDENCE -> 豐水薄野站或大通轉車到環狀通東站 -> 巴士到莫埃來沼公園 -> 午餐/輕食 -> 回市區 -> 福住站轉巴士到羊之丘展望台 -> 回大通 -> 夏祭/盆踊/狸小路晚餐逛街 -> 步行/地下鐵回旅館。  

若不想同一天跑兩個郊區點：保留莫埃來沼公園，羊之丘改成 Day 4 或取消，晚上早點回大通。

#### 1. Hotel ONE'S RESIDENCE -> 環狀通東站 -> 莫埃來沼公園東口

這天移動較分散，早上先去莫埃來沼公園比較舒服。從旅館可走到豐水薄野站或到大通轉車，再到環狀通東站轉巴士。

#### 2. 莫埃來沼公園

莫埃來沼公園由雕刻家野口勇設計，整座公園像一件大型地景作品，面積很大，有玻璃金字塔、人工山、噴泉、大片草地。8 月天氣好時很漂亮，但也要注意防曬與補水。

必吃：公園內午餐/輕食或自備飲料點心。這裡重點是景觀與散步，不要把正餐期待放太高。

#### 3. 福住站 -> 羊之丘展望台

下午回市區後轉往福住站，再搭巴士到羊之丘展望台。若上午在莫埃來沼停留太久，這站可以取消，避免晚上錯過夏祭收尾。

#### 4. 羊之丘展望台

羊之丘展望台有克拉克博士像、牧草地與札幌市景，夏天景色開闊，也能吃到成吉思汗料理。這裡停留不用太久，適合當作下午視野開闊的景點。

必吃：羊之丘展望台或札幌市區的成吉思汗烤羊肉。

#### 5. 大通公園 / 狸小路

2026 年北海盆踊到 8/16，剛好是最後一晚，很值得保留。晚上回大通，感受夏祭、盆踊和大通啤酒花園氣氛，再慢慢走回狸小路或薄野。

必吃：大通啤酒花園/夏祭攤位、狸小路周邊拉麵、湯咖哩、居酒屋。雨天也方便，因為狸小路有屋頂。

## Day 6 - 2026/8/17（一）旭川一日遊

### 建議路線

Hotel ONE'S RESIDENCE -> 中島公園站 -> 札幌站搭 JR 特急 -> 旭川站 -> 上野農場 -> 旭川拉麵村 -> 平和通買物公園 / 旭川市區散步 -> JR 特急回札幌 -> 地下鐵或計程車回旅館。  

旭川這天以花園、市區與拉麵為主。如果當天天氣太熱或下雨，可把上野農場縮短，增加旭川站周邊室內休息時間。

#### 1. Hotel ONE'S RESIDENCE -> JR 札幌站

早上先從中島公園站接到札幌站，搭 JR 特急 Lilac/Kamui 往旭川。這天仍是長距離日歸，建議前一晚先確認車次與指定席需求。

#### 2. JR 旭川站

旭川站是當日交通核心，到站後可依天氣決定先去上野農場或先吃飯。車站周邊也適合買飲料、點心與回程車上補給。

#### 3. 上野農場

上野農場是夏季花園候選，適合看北海道庭園、花草與季節景觀。節奏較慢，對二大一青少年的一日遊來說會比較舒展。

必吃：上野農場咖啡/甜點。若現場營業狀況許可，可作為下午茶；若未營業，改回旭川市區咖啡店。

#### 4. 旭川拉麵村

旭川拉麵村集合多家旭川代表拉麵店，旭川拉麵通常以醬油系、油脂封住熱度的湯頭聞名。這裡很適合作為旭川日歸的主餐。

必吃：旭川醬油拉麵。八家知名拉麵店聚集，適合依排隊與口味選，不必鎖死單一店家。

#### 5. 平和通買物公園 / 旭川市區

平和通買物公園是旭川市中心步行街，適合回程前散步、買飲料、找咖啡或甜點。若上野農場停留較短，這裡可以多放一點時間。

必吃：旭川市區甜點/咖啡、JR 車站便當。回札幌車上可準備飲料與點心，JR 北海道特急車上通常不要期待完整販售服務。

## Day 7 - 2026/8/18（二）登別溫泉與地獄谷一日遊

### 建議路線

Hotel ONE'S RESIDENCE -> 中島公園站 -> 札幌站 -> JR 特急到登別站 -> 巴士到登別溫泉巴士總站 -> 地獄谷步道 -> 大湯沼/天然足湯（視天氣） -> 極樂通商店街午餐 -> 登別伊達時代村 -> 溫泉街短暫泡湯或買伴手禮 -> 巴士回登別站 -> JR 回札幌 -> 地下鐵或計程車回旅館。  

2026 年鬼花火 8 月沒有列入官方場次，登別地獄祭也在你們離開後才開始；建議傍晚回札幌，避免太晚轉車。

#### 1. Hotel ONE'S RESIDENCE -> JR 札幌站 -> JR 登別站

這天從札幌日歸登別，交通時間較長。早上先接到札幌站，搭 JR 特急到登別站後，再轉巴士到登別溫泉巴士總站。

#### 2. 登別溫泉巴士總站 / 溫泉街

登別溫泉街是當日步行核心，地獄谷、極樂通商店街、日歸泡湯與伴手禮都在這一帶。先確認回登別站的巴士時間，會讓後半天比較安心。

#### 3. 地獄谷

登別是北海道代表性溫泉地，地獄谷是火山地形形成的蒸氣谷地，硫磺氣味、白煙與岩壁很有張力。從溫泉街步行即可到地獄谷，是這天最重要的必看景點。

#### 4. 大湯沼 / 大湯沼川天然足湯

若體力與天候允許，可從地獄谷延伸到大湯沼與天然足湯。步道狀況受天氣影響，若下雨或路面濕滑，就縮短為地獄谷周邊散步。

必吃：溫泉饅頭、牛奶、霜淇淋。適合作為散步小吃。

#### 5. 極樂通商店街

極樂通商店街是登別溫泉街主要用餐與購物區，午餐安排在這裡最順。逛街時也可看登別鬼與閻魔相關元素。

必吃：登別閻魔炒麵、溫泉市場 Onsen Ichiba、地獄拉麵。閻魔炒麵以北海道小麥平麵、特製醬與在地食材為特色；溫泉市場以新鮮海鮮、炭烤與在地料理聞名；地獄拉麵適合喜歡辣味的人，但要依青少年可接受辣度調整。

#### 6. 登別伊達時代村

登別伊達時代村是江戶時代主題村，有表演與場景，青少年可能比較有參與感。下午可把時間留給伊達時代村或溫泉街慢慢走。

#### 7. 溫泉街短暫泡湯 / 返回札幌

若時間允許，可在溫泉街安排日歸泡湯或買伴手禮。之後依巴士時間回 JR 登別站，再搭 JR 回札幌，最後從札幌站轉地下鐵或計程車回 Hotel ONE'S RESIDENCE。

## Day 8 - 2026/8/19（三）札幌退房、新千歲機場返台

### 建議路線

早餐 -> 退房 -> 若旅館可寄放行李，可在薄野/大通附近最後採買 1 小時；若不想冒險，直接從旅館到札幌站，再搭 JR 前往新千歲機場 -> 機場午餐與伴手禮 -> 15:05 CI131 起飛。  

建議至少 12:00 前抵達新千歲機場。國際線加上暑假期間，機場採買與託運都可能花時間。

#### 1. Hotel ONE'S RESIDENCE / 中島公園站

早上先完成退房。若旅館可寄放行李，可以在薄野、大通或札幌站周邊做最後採買；若不確定寄放服務，建議直接往機場移動。

#### 2. JR 札幌站

從旅館到札幌站可搭地下鐵，也可行李多時叫計程車。JR 札幌站搭快速 Airport 前往新千歲機場，建議預留候車與找月台時間。

#### 3. 新千歲機場國際線航廈

建議至少 12:00 前抵達新千歲機場。國際線加上暑假期間，機場採買與託運都可能花時間；這天不建議在札幌市區排太滿。

必買/必吃：Royce、六花亭、北菓樓、白色戀人、LeTAO、札幌農學校、北海道牛奶、霜淇淋、玉米零食、薯條三兄弟。若前面沒吃夠，可在機場拉麵道場補一餐。
